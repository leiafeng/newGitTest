# newGitTest

1. Clone repository
```
  git clone <link>
```

2. Check on status
```
  git status
```

3. Add file
```
  git add <file>
```

4. Add all changes
```
  git add -A
```

5. Commit with message
```
  git commit -m "message"
```

6. Push to GitHub
```
  git push
```

7. Pull from GitHub
```
  git pull
```
